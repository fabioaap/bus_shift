## 9. Requisitos Técnicos

### 9.1 Engine/Framework
- **Game Engine:** Unity 6000.3.2f1
- **Linguagem:** C# (8.0+)
- **IDE:** Visual Studio 2022+
- **Pipeline:** URP (Universal Render Pipeline)

### 9.2 Performance Targets
- FPS: 60+ (target mínimo)
- Resolução: 1920x1080 (1440p suportado)
- Loading Times: <5s entre cenas
- Memory: <2GB RAM
- Draw Calls: <100 por frame
